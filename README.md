# SOLO DORK

**SOLO DORK** is an advanced Google Dork query builder built for cybersecurity education, OSINT research, and authorized security assessments. It provides a clean, modern interface for generating Google search queries using common search operators such as `site:`, `filetype:`, `intitle:`, `inurl:`, `intext:`, and more.

## ✨ Features

* Advanced Google Dork query builder
* 30+ Google search operators
* Predefined OSINT query templates
* One-click copy to clipboard
* Responsive, modern UI
* Fast and lightweight
* GitHub Pages compatible
* No external dependencies

## 🛠️ Built With

* HTML5
* CSS3
* JavaScript (Vanilla)

## 🎯 Intended Use

SOLO DORK is designed for:

* Cybersecurity education
* OSINT research
* Learning Google search operators
* Authorized reconnaissance
* Security awareness and training

# Installation

## Clone the Repository

```bash
git clone https://github.com/heyneeraj/solo-dork.git
```

## Navigate to the Project Directory

```bash
cd solo-dork
```

## Make the Launcher Executable

```bash
chmod +x solodork.sh
```

## Launch SOLO DORK

```bash
./solodork.sh
```

---

## Requirements

* Linux (Kali Linux recommended)
* Bash
* A modern web browser (Firefox, Chromium, or Google Chrome)

---

## Troubleshooting

If you receive a **Permission denied** error, run:

```bash
chmod +x solodork.sh
```

If the launcher still does not execute, try:

```bash
bash solodork.sh
```
